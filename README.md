Easy Cloudflare R2 Storage
WordPress插件，用于将媒体文件迁移到Cloudflare R2存储。

功能特性
自动上传新媒体文件到R2

URL重写（本地URL替换为R2 URL）

批量同步现有文件

支持图片所有尺寸（缩略图等）

定时自动同步任务

详细的调试日志

安装使用
上传插件并激活

在【设置】-【Easy R2 Storage】中配置：

账户ID
访问密钥ID
秘密访问密钥
存储桶名称
测试连接并保存设置

根据需要启用自动上传和URL重写

配置说明
必要配置
R2账户凭据（从Cloudflare控制台获取）

存储桶名称（需要在R2中提前创建）

可选配置
自动上传：启用后新上传文件自动传到R2

URL重写：将媒体URL替换为R2地址

删除本地文件：上传成功后删除服务器文件（谨慎启用）

调试日志：记录详细操作日志用于排错

注意事项
R2免费额度：10GB存储 + 1000万次操作/月

首次使用建议先测试连接和小批量同步

启用"删除本地文件"前确保URL重写正常工作

大站建议分批同步，避免服务器负载过高
